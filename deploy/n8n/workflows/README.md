# n8n Workflows Backup

이 디렉토리는 n8n 워크플로우 JSON 파일을 백업하기 위한 공간입니다.

## 백업 방법 (Manual)
1. n8n UI (http://localhost:5678) 접속
2. 백업할 워크플로우 열기
3. 메뉴 -> Export -> Download to File
4. 다운로드된 JSON 파일을 이 디렉토리에 저장 (예: `daily-report-workflow.json`)

*참고: 현재 자동화된 백업 프로세스는 구축되어 있지 않으므로, 주요 변경 사항이 있을 때마다 수동 백업을 권장합니다.*
